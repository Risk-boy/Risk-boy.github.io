## :pencil2:코드

```python
import sys
sys.stdin = open("input.txt")
input = sys.stdin.readline


n = int(input())
p = 10 ** 9 + 7
dp = [0] * (10 ** 6)
dp[0] = 1
dp[1] = 2
x = sum(dp[:2])
for i in range(2, n + 1):
    dp[i] = 2 * x + dp[i - 2]
    x += dp[i]

print(dp[n])

```

## :star:풀이

풀이는 그림으로!! 

n = 3부터 생기는 지그재그타일 2개를 잘 체크해주고 시간초과가 걸리지 않게 prefix_sum을 구해 dp에 더해주는 방식으로 풀면된다:smile:
